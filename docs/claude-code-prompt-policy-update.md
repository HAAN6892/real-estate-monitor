# Claude Code 작업 프롬프트 — 정책 타임라인 업데이트

## 배경
대시보드 정책 탭의 "최근 정책 변동" 타임라인에 누락/오류 항목이 있음.
2025.03~2026.03 사이 정책 변경사항을 최신화해야 함.

## 작업 대상 파일
app.js (정책 타임라인 데이터가 하드코딩된 부분)

## 수정 내용

### 1. 기존 항목 수정

#### (1) "2026.02.26 기준금리 동결 5회 연속" → 6회 연속으로 수정
- "5회 연속" → "6회 연속"
- 나머지 내용 동일

#### (2) "2025.01 신생아 특례대출 한도 상향" 항목의 우리 상황 영향 업데이트
- 현재: "출산 시 합산 8,740만 < 2억 → 자격 충족, 한도 최대 2.5억으로 대폭 상향"
- 수정: "출산 시 합산 8,740만 < 2억 → 자격 충족, 한도 최대 2.5억→4억. 단 2025.7월 이후 계약분은 한도 4억, 수도권 LTV 70%로 축소됨"

### 2. 새로 추가할 항목 (시간순 — 기존 항목 사이에 삽입)

#### (1) 2025.03.24 — 신생아 특례 금리 인상 + 실거주 강화
```
날짜: 2025.03
제목: 신생아 특례 금리 인상 + 디딤돌 실거주 2년
설명: 특례금리 1.6~4.3% → 1.8~4.5%로 인상
      디딤돌대출 실거주 의무 1년 → 2년으로 강화
영향색: 주의(노란색/주황색)
우리 상황 영향: 출산 후 신생아 특례 활용 시 금리 부담 소폭 증가. 실거주 2년 의무 → 전세 놓기 어려움.
```

#### (2) 2025.07.01 — 스트레스 DSR 3단계 시행
```
날짜: 2025.07
제목: 스트레스 DSR 3단계 시행
설명: 전 업권 모든 가계대출에 스트레스 금리 1.5% 적용
      수도권 규제지역은 10.15 대책으로 3.0% 적용 중
      지방 주담대는 0.75%로 유예 (→ 2026.6.30까지 연장)
영향색: 주의(노란색/주황색)
우리 상황 영향: DSR 40% 기준 대출한도 실질적 축소. 변동금리 선택 시 한도 약 11% 감소. 고정금리 선택이 유리.
```

#### (3) 2025.07.04 — 신생아 특례 한도 축소 + LTV 강화
```
날짜: 2025.07
제목: 신생아 특례 한도 축소 (5억→4억)
설명: 구입대출 한도 최대 5억 → 4억으로 축소
      수도권·규제지역 LTV 70% (생애최초도 동일)
      2025.6.28 이후 매매계약 체결분부터 적용
영향색: 주의(노란색/주황색)  
우리 상황 영향: 출산 후 신생아 특례 활용해도 수도권 최대 4억 한도. 생애최초 LTV 80% 혜택 사라짐 (Hans는 어차피 해당 없음).
```

#### (4) 2025.12 — 지방 주담대 ST DSR 유예 연장
```
날짜: 2025.12
제목: 지방 주담대 스트레스 DSR 유예 6개월 연장
설명: 지방(서울·경기·인천 제외) 주담대 스트레스 금리
      3단계(1.5%) 적용을 2026.6.30까지 유예
      2단계(0.75%) 유지
영향색: 정보(파란색)
우리 상황 영향: 수도권 매수 타겟이므로 직접 영향 없음. 대구 매도에는 간접 긍정(지방 매수 수요 유지).
```

#### (5) 2026.01 — 신생아 특례 순자산 기준 상향
```
날짜: 2026.01
제목: 신생아 특례 순자산 기준 상향 (4.88억→5.11억)
설명: 2026년 자산심사 기준 변경
      순자산가액 4.88억 → 5.11억으로 상향
      청약통장 우대금리 기준 강화, 중도상환수수료 인하
영향색: 긍정(초록색)
우리 상황 영향: 자산 기준 완화로 출산 시 자격 충족 가능성 높아짐.
```
